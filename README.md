## ✨ Features

- **Dark Futuristic Design** — Charcoal, silver, and violet palette with glassmorphism cards
- **Scroll Animations** — Smooth reveal effects powered by IntersectionObserver
- **Fully Responsive** — Pixel-perfect on desktop, tablet, and mobile
- **Conversion-Focused** — Strategic CTA placement designed to turn visitors into clients
- **Fast & Lightweight** — Pure HTML, CSS, and vanilla JS — no frameworks, no bloat
- **SEO Optimized** — Semantic HTML, meta tags, and clean heading structure
- **Accessible** — Proper ARIA labels, keyboard navigation, and reduced-motion support

## 🛠 Tech Stack

| Layer | Technology |
